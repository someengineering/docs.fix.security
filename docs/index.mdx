# Introduction

**Welcome to the documentation for [Fix by Some Engineering Inc.](https://fix.security)!**

This documentation is intended to help you understand how to use the Fix platform and its various features.

## About Fix

**Fix is a <abbr title="Cloud Security Posture Management">CSPM</abbr> tool.** Fix performs routine compliance checks against snapshots of your infrastructure, identifies misconfigurations, and provides recommendations and workflows to correct detected issues.

Learn more about Fix at [https://fix.security](https://fix.security).
